# Digital Diachrony 2025 - Workshop Website

Beautiful, modern website for the Digital Diachrony 2025 workshop in Delphi, Greece.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
